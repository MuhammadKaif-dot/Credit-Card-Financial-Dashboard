# Credit-Card-Financial-Dashboard
A Credit Card Financial Dashboard built to uncover hidden trends and patterns in credit card operations. It provides weekly performance insights, key financial metrics, and real-time analytics for better decision-making. Data is sourced from MySQL and visualized in Power BI using interactive dashboards and filters.

# Credit Card Financial Dashboard

This dashboard uncovers hidden trends and patterns in credit card operations using weekly performance metrics and real-time analytics.  
Data is sourced from MySQL and visualized in Power BI.

## Tech Stack
- MySQL (Data Source)
- Power BI (Data Visualization & Reporting)

## Features
- Weekly performance monitoring
- Trend & pattern analysis
- Real-time KPI insights
